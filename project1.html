Project 1: Analysis on the Spread of Tuberculosis

End-to-End Data Science Pipeline

Comprehensive analysis of global tuberculosis spread patterns from 2007-2014 using WHO data. This project implements a complete data science pipeline including data acquisition via Kaggle API, NoSQL database integration, statistical modeling, big data processing with PySpark, and deep learning predictions using TensorFlow. Analyzed 1,548 records across 194 countries to identify mortality trends and geographic disparities.

Technologies & Libraries:

Python Pandas NumPy Matplotlib Seaborn Plotly Statsmodels PySpark TensorFlow/Keras Scikit-learn TinyDB Kaggle API Flask

Technical Concepts Demonstrated:

  • Data Acquisition & Ingestion: Automated dataset download using Kaggle API with authentication, file system navigation, and NoSQL database integration with TinyDB for structured JSON storage
  • Data Cleaning & Preprocessing: Handling missing values (20 records imputed using median strategy), data type conversion with regex pattern matching, removal of non-numeric characters from formatted strings
  • Feature Engineering: Created year-over-year change metrics, calculated uncertainty ratios from confidence interval ranges, and generated derived features for improved model performance
  • Exploratory Data Analysis (EDA): Statistical profiling with descriptive statistics, null value analysis, and data distribution assessment across 14 features
  • Statistical Modeling: OLS regression analysis using Statsmodels with R² = 0.836, examining relationships between TB prevalence and mortality while controlling for temporal trends
  • Big Data Processing: Apache Spark implementation with custom schema definition, distributed data transformations, GroupBy aggregations, and DataFrame filtering operations
  • Machine Learning (Spark MLlib): Linear regression model with VectorAssembler for feature engineering, achieving RMSE = 0.624 and R² = 0.999 on test data (80/20 train-test split)
  • Deep Learning: Feedforward neural network with 2 hidden layers (64 and 32 neurons), ReLU activation, Adam optimizer, achieving test MAE = 0.446 over 50 epochs with StandardScaler normalization
  • Data Visualization: Static visualizations (histograms, line plots, bar charts) and interactive Plotly dashboards with hover functionality for multi-dimensional exploration
  • Model Deployment: Flask API development with REST endpoint for real-time predictions, model serialization (HDF5 format), and StandardScaler persistence using joblib

Analytical Methods:

  • Regression Analysis: Multivariate OLS regression with F-statistic = 3932, examining TB prevalence and temporal effects on mortality rates
  • Time Series Analysis: Trend analysis revealing 22% reduction in average TB mortality (18 to 14 deaths per 100k) from 2007-2014
  • Geographic Analysis: Country-level comparative analysis identifying 15 highest-burden nations, with 11 in Africa showing mortality rates 4-7x global average
  • Distribution Analysis: Right-skewed mortality distribution analysis revealing global inequality in TB burden
  • Feature Scaling: StandardScaler implementation for neural network input normalization
  • Model Evaluation: Multiple metrics including RMSE, R², MAE, and MSE for comprehensive model performance assessment

Key Insights:

  • Declining Global Trend: TB mortality decreased 22% from 2007-2014 (18 to 14 deaths per 100k population), with steeper decline 2007-2009 followed by gradual reduction, suggesting initial program success with persistent challenges
  • Geographic Disparity: 11 of 15 highest-mortality countries located in sub-Saharan Africa, with Djibouti showing 95 deaths per 100k—nearly 7x the 2014 global average—highlighting severe regional inequality
  • Strong Prevalence-Mortality Correlation: OLS regression revealed highly significant relationship (p < 0.001) between TB prevalence and mortality with R² = 0.836, indicating prevalence as strong predictor of death rates
  • Skewed Distribution: Right-skewed mortality distribution shows most countries achieving low rates while outliers persist, indicating uneven progress in TB control efforts
  • Predictive Model Performance: Deep learning model achieved MAE = 0.446 deaths per 100k, demonstrating high accuracy in mortality prediction based on prevalence, temporal, and confidence interval features
  • Data Quality Observations: Missing values concentrated in confidence interval ranges (20 records, 1.3%), suggesting uncertainty in estimation methodologies for certain countries/years
  • Temporal Plateau Effect: Mortality reduction slowed 2012-2013, potentially indicating diminishing returns from existing interventions and need for targeted strategies in
